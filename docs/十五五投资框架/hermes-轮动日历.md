## 八、季节性操作日历

### 每年固定节奏

```
月份   特征               操作策略
────────────────────────────────────────────
1月    年报预告+春季躁动    布局两会+十五五主线
2月    春节前后+两会预期    持仓或加仓政策主线
3月    两会召开             逢高减仓（利好兑现）
4月    年报+一季报密集       验证业绩，优胜劣汰
5月    业绩真空期           减少操作，等待催化
6月    年中布局窗口         低位布局下半年主线
7月    中报预告+夏季行情    储能/电力旺季
8月    中报密集期          验证主线逻辑
9月    秋季行情             信创招标+数据要素
10月   三季报+中央经济会议前瞻 调整仓位
11月   跨年行情预热         布局来年方向
12月   中央经济工作会议+收官  兑现盈利，布局新周期
```
