# 十五五规划政策总纲

> 基于《十五五规划纲要》（2026-03-13）的政策解读与投资总纲
> 2026年6月 整理 | 核心逻辑：新质生产力 × 高质量发展 × 安全与发展平衡

---

## 一、宏观定位：十五五的底层逻辑

### 1.1 历史坐标

| 阶段 | 规划 | 核心特征 |
|------|------|---------|
| 十二五(2011-15) | 战略性新兴产业 | 粗放增长末期，基建地产驱动 |
| 十三五(2016-20) | 供给侧改革 | 去产能、去杠杆 |
| 十四五(2021-25) | 双循环+碳中和 | 新能源产业链爆发期 |
| **十五五(2026-30)** | **新质生产力** | **质量效率型增长，数字经济驱动** |

### 1.2 三条底层逻辑变化

**① 考核指标变了**
- 不再提"半导体自给率70%"→ 改考"**数字经济占GDP 12.5%**"
- 从"造多少"到"用多深"——算力渗透率是核心KPI
- 核心框架从"追赶先进制程" → **模芯云用**（模型-芯片-云-应用）四位一体

**② 产业打法变了**
- 新能源/电动车已验证成功 → 复制到**半导体、生物技术、量子**三大新领域
- 制造业主导 → **数字经济与服务型制造**并重

**③ 安全权重变了**
- 六大原则之一：平衡安全与发展
- 军民两用供应链自主可控写入正文
- 粮食安全、能源安全均为**约束性**指标

### 1.3 宏观含义
- GDP 4.5-5% → **结构性机会 > 总量机会**，普涨难、分化大
- 支持企业 > 再分配 → **科技龙头利润率有望扩张**
- 财政方向：从基建投资转向 **产业扶持 + 研发补贴**

## 二、关键量化目标（约束性指标是核心锚）

| 指标 | 2025基准 | 2030目标 | 性质 | 对应投资方向 |
|------|---------|---------|------|------------|
| 数字经济核心产业/GDP | 10.2% | **12.5%** | 预期性 | AI数字中国、信创 |
| R&D经费年均增长 | — | **7.1%** | 预期性 | 半导体设备、创新药 |
| 高价值发明专利/万人 | 15.3 | **>22** | 预期性 | 科技自主可控 |
| **非化石能源占比** | 20.8% | **25%** | **约束性** | 新能源、储能、电网 |
| **单位GDP碳排放下降** | — | **-17%** | **约束性** | 碳交易、节能降碳 |
| **粮食综合产能** | 1.39万亿斤 | **>1.45万亿斤** | **约束性** | 种业、农业科技 |
| **能源总产能** | 48亿吨标煤 | **58亿吨标煤** | **约束性** | 能源安全、油气装备 |
| GDP增速 | — | 4.5-5% | 预期性 | 全市场基准 |

> 约束性指标是十五五的"军令状"，对应方向确定性最高

## 三、三大投资主线 + 辅助线

```
主线A：数字中国 ─── 算力基建 → 信创 → AI应用 → 数据要素
主线B：能源革命2.0 ─ 储能 → 电网 → 氢能 → 碳交易
主线C：科技自主可控 ─ 半导体设备 → 创新药 → 量子技术
辅助线：安全保障 ─── 军工 → 种业 → 跨境支付
```

### ⚡ 主线A：数字中国

| 维度 | 内容 |
|------|------|
| **政策锚** | 数字经济核心产业/GDP 10.2% → **12.5%** |
| **时间跨度** | 2026-2030（全程受益） |
| **子方向** | 算力基建、信创替代、数据要素、AI应用 |
| **关键节点** | 2026H2信创招标 → 2027算力运营 → 2028-30 AI应用变现 |

### ☀️ 主线B：能源革命2.0

| 维度 | 内容 |
|------|------|
| **政策锚** | 非化石能源占比 20.8% → **25%（约束性）** |
| **时间跨度** | 2026-2030（前两年放量，后三年冲刺） |
| **子方向** | 光伏/风电、新型储能、智能电网、氢能、碳交易 |
| **关键判断** | 光伏龙头集中度提升 → 储能商业模式成型 → 电网确定性最高 |

### 🔬 主线C：科技自主可控

| 维度 | 内容 |
|------|------|
| **政策锚** | R&D经费年均增长 **7.1%** |
| **时间跨度** | 2026-2030（逐年加速） |
| **子方向** | 半导体设备/材料、先进封装、创新药、量子技术 |
| **关键判断** | 半导体设备"政策兜底" → 创新药2026-28黄金三年 → 量子为远期期权 |

### 🛡️ 辅助线：安全保障

| 子方向 | 逻辑 | 弹性/确定性 |
|--------|------|------------|
| 军工电子/特种芯片 | 军民两用供应链安全 | ★★★ / ★★★★★ |
| 种业/粮食安全 | 约束性指标1.45万亿斤 | ★★ / ★★★★★ |
| 跨境支付/数字人民币 | 人民币国际化+CIPS | ★★★ / ★★★ |
| 信创安全/密码 | 数据安全法落地 | ★★★ / ★★★★ |

## 五、选股方法论

### 四个选股范式

| 范式 | 适用场景 | 核心指标 | 仓位策略 |
|------|---------|---------|---------|
| **龙头确定性** | 成熟赛道（光伏、电网、信创） | 市占率>15%、ROE>12%、经营现金流为正 | 核心底仓 |
| **隐形冠军** | 细分环节（材料、零部件） | 市占率>30%、毛利率>40%、客户依赖<30% | 中仓持有 |
| **困境反转** | 周期底部（光伏、存储） | PB历史低位、产能出清信号、在手订单回升 | 波段操作 |
| **0到1期权** | 前沿领域（量子、氢能、生物制造） | 技术壁垒、政府订单、专利数量 | 轻仓<5% |

### 财报验证三件套
1. **营收增速 > 行业平均** → 份额在提升
2. **毛利率同比不降** → 定价权/成本控制有效
3. **应收账款周转率不降** → 收入质量好

## 六、仓位管理（1/3/7体系）

| 大盘状态 | 总仓位 | 数字中国 | 能源革命 | 科技自主 | 安全 | 现金 |
|---------|-------|---------|---------|---------|------|------|
| 🟢 上升趋势 | **70%** | 25% | 20% | 15% | 10% | 30% |
| 🟡 横盘震荡 | **30%** | 12% | 8% | 5% | 5% | 70% |
| 🔴 下跌趋势 | **10%** | 4% | 3% | 2% | 1% | 90% |

### 动态调整原则
- **季末再平衡**：超配>5%则卖出，低配>5%则买入
- **事件驱动**：重大政策/财报 → 第一时间调整
- **止损纪律**：单票-15%无条件止损

## 七、每年固定操作节奏

```
月份   特征               操作策略
────────────────────────────────────────────
1月    年报预告+春季躁动    布局两会+十五五主线
2月    春节前后+两会预期    持仓或加仓政策主线
3月    两会召开             逢高减仓（利好兑现）
4月    年报+一季报密集       验证业绩，优胜劣汰
5月    业绩真空期           减少操作，等待催化
6月    年中布局窗口         低位布局下半年主线
7月    中报预告+夏季行情    储能/电力旺季
8月    中报密集期           验证主线逻辑
9月    秋季行情             信创招标+数据要素
10月   三季报+中央经济前瞻   调整仓位
11月   跨年行情预热         布局来年方向
12月   中央经济工作会议+收官  兑现盈利，布局新周期
```

### 买卖节奏口诀
> **春季政策催，别等两会开**  
> **四月看业绩，假的别硬扛**  
> **五六月布局，七八月收获**  
> **九月信创起，十月数据热**  
> **年末看跨年，三年规划前**
