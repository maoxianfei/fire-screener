# 十五五规划投资框架

> 基于《中华人民共和国国民经济和社会发展第十五个五年规划纲要》（2026-2030）的政策投资框架
> 创建日期：2026-06-24 | 状态：骨架搭建中，逐步填充
> 覆盖范围：A 股为主

---

## 框架方法论

```
十五五规划政策分析
     ↓
  投资方向（8条主线）
     ↓
 ┌─────────────────────────────────────┐
 │           投资框架矩阵               │
 │  ├── 产业（一级分类）              │
 │  ├── 板块（二级细分）              │
 │  ├── 核心概念/标的池（三级）       │
 │  ├── 短期兑现目标（< 1年）         │
 │  ├── 中长期目标（1-5年）           │
 │  ├── 风险监测指标                  │
 │  └── 事件驱动日历                  │
 └─────────────────────────────────────┘
     ↓
  月度板块轮动策略
  （每月更新，叠加政策/技术催化剂判断）
```

## 框架使用指引

### 每周/每月更新流程
1. 查阅 **[[事件日历]]** — 未来1-3月政策/技术/事件节点
2. 评估 **[[风险监测]]** — 当前各方向风险信号
3. 查看 **[[月度轮动]]** — 本月推荐的板块配置
4. 深入具体方向页面 → 选择标的 → 结合技术分析择时

### 框架更新规则
- **每月初**：更新月度轮动策略
- **政策发布时**：更新事件日历，调整方向优先级
- **重大技术突破时**：更新具体方向的技术兑现进度
- **财报季**：更新板块基本面信号

### 与 DSA 系统的配合
- 本框架提供 **政策/宏观/产业逻辑** 支撑
- DSA（daily_stock_analysis）提供 **技术面/数据/个股分析**
- 最终决策 = 框架方向 + 技术择时 + 风险控制
