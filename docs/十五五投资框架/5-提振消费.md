# 方向5：提振消费

> 十五五规划：第五篇第十五章「大力提振消费」

## 一、政策依据

- **第十五章 大力提振消费**
  - 第一节：夯实居民消费基础 — 稳就业、促增收、稳预期、楼市股市健康发展
  - 第二节：释放服务消费潜力 — 养老/托育/文旅/餐饮住宿/家政
  - 第三节：促进大宗消费 — 汽车/家电以旧换新、电子产品消费
  - 第四节：培育壮大新型消费 — 数字消费/绿色消费/健康消费/智能消费
  - 第五节：强化消费保障 — 消费环境/信用体系/消费者权益

## 三、兑现节奏

### 短期（2026H2 - 2027H1）
- **以旧换新政策**：家电/汽车补贴续期→销量数据验证
- **节假日消费**：暑期/国庆/元旦→文旅/零售催化
- **AI 终端换机**：AI PC/手机渗透率快速提升

### 中期（2027-2028）
- **银发经济**：养老服务体系政策落地→产业规模化
- **消费基础设施**：县域商业体系、一刻钟便民生活圈
- **收入分配改革**：中等收入群体扩大→消费升级

## 四、风险监测

| 风险指标 | 监测信号 | 影响程度 |
|---------|---------|---------|
| 居民收入增速放缓 | 社零/可支配收入持续走弱 | 🔴 高 |
| 以旧换新政策退坡 | 补贴退坡后销量回落 | 🟡 中 |
| 房地产财富效应恶化 | 房价持续下跌→消费信心弱 | 🔴 高 |
| 就业压力 | 失业率上升/就业景气度下行 | 🔴 高 |
| 消费降级加速 | 客单价持续下降 | 🟡 中 |
